## react-next-demo

- React ^17 + Next.js ^10 + TypeScript ^4
- MaterialUI ^5
- Prettier + ESlint + AirBnb + Husky
- Redux + Redux-toolkit
- Plop.js scripts

## Getting Started

First, run the development server:

```bash
yarn dev
```

## Create a new component

```bash
yarn plop component NewComponent
```

see `./plop-templates/component`
